# We Buy Any Car Florida

This is a simple web page for making appointments, built with Vue and Express. It uses a MySQL database to store the data.


## Project management

https://www.pivotaltracker.com/n/projects/2682005

## Project setup

```
cp .env.example .env
npm install
```

## Running the app

```
npm start
```
